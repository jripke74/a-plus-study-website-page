1.0 Mobile Devices

  1. Given a scenario, monitor mobile devices hardware and use appropriate replacement techniques.
  2. Compare and contrast accessories and connectivity options for mobile devices.
  3. Given a scenario, configure basic mobile device network connectivity and provide application support.

2.0 Networking

  1. Compare and contrast Transmission Control Protocol (TCP) and User Datagtram Protocol (UDP) ports, protocols, and their purposes.
  2. Explain wireless networking technologies.
  3. Summarize service provided by networked hosts.
  4. Explain common network configuration concepts
  5. Compare and contrast common networking hardware devices.
  6. Given a scenario, configure basic wired/wireless small office/home office (SOHO) networks.
  7. Compare and contrast internet connection types, network types, and their characteristics.
  8. Explain networking tools and their purposes.

3.0 Hardware

  1. Compare and contrast display components and attributes.
  2. Summarize basic cable types and their connectors, features, and purposes.
  3. Compare and contrast RAM characteristics.
  4. Compare and contrast storage devices.
  5. Given a scenario, install and configure motherboards, central processing units (CPUs), and add-on cards.
  6. Given a scenario, install the appropriate power supply.
  7. Given a scenario, deploy and configure multifunction devices/printers and settings.
  8. Given a scenario, perform appropriate printer maintenance.

4.0 Virtualization and Cloud Computing

  1. Explain virtualization concepts.
  2. Summarize cloud computing concepts.

5.0 Hardware and Networking Troubleshooting

  1. Given a scenario, troubleshoot motherboards, RAM, CPUs, and power.
  2. Given a scenario, troubleshoot drive and RAID issues.
  3. Given a scenario, troubleshoot video, projector, and display issues.
  4. Given a scenario, troubleshoot common mobile device issues.
  5. Given a scenario, troubleshoot network issues.
  6. Given a scenario, troubleshoot printer issues.
